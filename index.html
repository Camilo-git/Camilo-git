Acerca de MI

Soy un apacionado tecnologico, muy interesado en el area de la programacion WEB Manejo con fluides lenguajes de maquetado como Html y CSS y algunas librerias como son Bootstrap Sass o maquetado con css Grid.
En la seccion de control de versiones uso Git y tengo en mi computador WSL (windows subsystem for linux) Para la practica del entorno con linux.

En aspectos de Back-end son de mi manejo lenguajes como JavaScript, Python y PHP como mi principal lenguaje de uso y coneccion a base de datos MYQL empleando Paradigmas como Programacion Orientada Objetos, Programacion Funcional, estructuras de datos
